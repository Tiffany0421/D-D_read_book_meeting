# read book meeting web

目前 react 使用 create-react-app 建構基礎框架

> 開發

開啟 webpack-dev-server 在 http://localhost:3000 上

```shell script
npm run start 
```

> 測試

之後可能會加入 Jest 當作測試工具

```shell script
npm run test 
```

> 編譯與部屬

```shell script
npm run build 
```


使用 material-ui 當做 UI 框架
